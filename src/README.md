# 指定意图话术配置后台 - 产品文档

## 产品概述

指定意图话术配置后台是一个用于管理客户服务场景下的智能话术系统。系统允许管理员针对不同的客户意图，配置相应的多段回复话术，每段话术支持文本、图片等多种形式的内容组合。

## 核心功能

### 1. 意图管理列表

意图管理列表是系统的主界面，提供了完整的意图配置管理功能。

#### 主要特性：
- **意图列表展示**：以表格形式展示所有已配置的客户意图
- **搜索筛选**：支持按意图名称快速搜索和筛选
- **批量操作**：支持导出意图配置数据
- **状态管理**：显示每个意图的启用/禁用状态

#### 列表字段说明：
| 字段名称 | 说明 |
|---------|------|
| 意图名称 | 客户意图的名称标识 |
| 描述 | 该意图的详细说明 |
| 话术数量 | 该意图下配置的话术段数 |
| 创建时间 | 意图配置的创建时间 |
| 更新时间 | 最后一次修改的时间 |
| 状态 | 启用/禁用状态 |
| 操作 | 编辑、删除等操作按钮 |

### 2. 意图话术编辑

点击列表中的"编辑"按钮，会打开意图话术编辑弹窗，这是系统的核心功能模块。

#### 基本信息配置：
- **意图名称**（必填）：为客户意图设置一个清晰的名称
- **描述**（可选）：添加对该意图的详细说明，方便团队成员理解

#### 多段话术配置：

系统支持为单个意图配置多段回复话术，这些话术会按顺序发送给客户。

**话术管理功能：**
- **添加话术**：点击"添加话术"按钮创建新的话术段
- **排序调整**：通过拖拽图标调整话术的发送顺序
- **内容编辑**：点击话术卡片进入编辑模式
- **删除话术**：移除不需要的话术段

**内容类型支持：**
每段话术支持多种内容形式的组合：
- 📝 **文本**：纯文本消息，最多1000个字符
- 🌐 **网页**：分享网页链接
- 🖼️ **图片**：上传或选择图片
- 🎬 **视频**：分享视频内容
- 📄 **文件**：上传文档、PDF等文件
- 📱 **小程序**：分享小程序卡片

**素材库集成：**
- 点击"引用素材库"可快速选择已上传的素材
- 支持文本模板、图片、视频等多种素材类型
- 提高话术配置效率，保证内容质量

### 3. 话术编辑器

话术编辑器提供了直观的内容编辑界面。

#### 编辑器特性：
- **标签页切换**：通过顶部标签快速切换内容类型
- **实时预览**：编辑过程中可实时查看话术效果
- **字数限制**：文本内容最多支持1000个字符
- **表情支持**：可插入emoji表情丰富话术内容
- **一键添加**：编辑完成后点击"添加"按钮保存当前话术段

## 使用流程

### 新增意图话术

1. 在列表页点击"新增意图"按钮
2. 在弹窗中输入意图名称和描述
3. 点击"添加话术"创建第一段话术
4. 选择内容类型（文本/图片/视频等）
5. 编辑话术内容
6. 点击"添加"保存当前话术段
7. 重复步骤3-6添加更多话术段
8. 点击"保存"完成配置

### 编辑现有意图

1. 在列表中找到需要编辑的意图
2. 点击该行的"编辑"按钮
3. 在弹窗中修改基本信息或话术内容
4. 可以添加新话术、编辑现有话术或删除话术
5. 通过拖拽调整话术顺序
6. 点击"保存"应用更改

### 删除意图

1. 在列表中找到需要删除的意图
2. 点击该行的"删除"按钮
3. 在确认对话框中确认删除操作
4. 系统将永久删除该意图及其所有话术配置

## 最佳实践

### 话术设计原则

1. **逻辑清晰**：按照信息传递的逻辑顺序组织话术段落
2. **内容精简**：每段话术专注于一个主题，避免信息过载
3. **图文并茂**：合理搭配文字和图片，提升信息传达效果
4. **及时更新**：定期检查和更新话术内容，保持信息时效性

### 命名规范

- 意图名称应简洁明了，准确反映客户需求类型
- 避免使用过于技术化的术语
- 建议使用中文命名，方便团队协作

### 内容管理

- 定期整理素材库，删除过时内容
- 对常用话术建立模板，提高配置效率
- 记录话术的使用效果，持续优化内容
